# 项目规格书："Hollywood Cut" (电影级 AI 幕后照生成器) - 爵士西部版

**角色：** 你是一名世界级的前端工程师和先锋 UI 设计师。
**目标：** 构建一个名为 "Hollywood Cut" 的单页 React 应用程序，利用 Google Gemini API 生成超写实的电影幕后片场照。

## 1. 技术架构
*   **框架:** React 19 (ES Modules, 无需构建步骤)。
*   **样式:** Tailwind CSS (通过 CDN)。
*   **动画:** Framer Motion (用于丝滑的过渡、加载和交互)。
*   **图标:** Lucide React。
*   **AI 集成:** `@google/genai` SDK。
*   **持久化:** 使用 `localStorage` 存储 API Key。

## 2. 视觉风格：“克莱因蓝 + 爵士西部 (Klein Blue Jazz Western)”
应用程序应呈现出一种**轻松、活泼、充满艺术气息**的氛围，融合了现代波普艺术的高饱和度与西部爵士的复古情调。

*   **配色方案:**
    *   **主色 (Core):** **克莱因蓝 (#002FA7)**。用于标题、边框、主要按钮和强调文字。象征艺术、深邃与现代。
    *   **背景 (Canvas):** **米白/纸张色 (#FAF9F6)**。作为纯净画布，无噪点，提供干净、温暖的阅读体验。
    *   **强调色 (Accent):** **日落橙 (#FF4500)**。用于“Action”按钮、悬停状态和关键高亮，带来活力与热情。
    *   **辅助色 (Detail):** **马鞍棕 (#8B4513)**。用于次要文字和细节，增添西部复古感。
*   **排版:**
    *   **标题:** 粗衬线字体 (`Playfair Display`, `Rye`)，加粗、大字号，类似老式电影海报或爵士乐传单。
    *   **正文:** 清晰的无衬线字体 (`Inter`)。
    *   **标签:** 等宽字体 (Monospace)，模拟打字机效果。
*   **氛围与质感:**
    *   **纯净背景:** 移除颗粒质感，采用平滑的暖色背景。
    *   **高对比度边框:** 使用粗线条和虚线边框，类似剪贴画或通缉令的设计语言。
    *   **装饰元素:** 背景中散落的爵士乐符、西部星星图案、抽象几何图形，以低透明度点缀。
    *   **动态背景:** 输入电影名后，生成的海报印象图以“正片叠底 (Multiply)”模式融合在背景中。

## 3. 用户流程与核心功能

### A. 欢迎/认证页面
*   **视觉效果:** 白色卡片悬浮在米色背景上，带有粗蓝边框和强烈的投影。
*   **文案:**
    *   **标题:** "HOLLYWOOD CUT" (克莱因蓝，特大号，Rye 字体)
    *   **副标题:** "开启您的好莱坞片场之旅" (马鞍棕，手写体感)
    *   **按钮:** 蓝色实心按钮，橙色悬停效果。

### B. 主应用布局
分屏设计，色彩明快。

#### 左侧面板：导演控制台
*   **UI 语言:** 白色卡片，蓝色粗边框，橙色阴影。
*   **第 1 部分：上传素材**
    *   白色背景，蓝色虚线边框。悬停时变为暖色调。
*   **第 2 部分：场景设定**
    *   输入框和按钮使用高对比度的蓝/白配色。选中状态使用日落橙高亮。
*   **第 3 部分：导演指令**
    *   文本区域模拟信纸效果，可能有折角设计。
*   **第 4 部分：ACTION**
    *   **按钮:** 巨大的橙色按钮，带有“ACTION!”字样和星星图标。点击时有强烈的反馈动画。

#### 右侧面板：监视器
*   **容器:** 模拟复古电视或取景框，蓝色粗边框。
*   **状态:**
    *   **空状态:** "NO SIGNAL" (无信号) 字样，波普艺术/西部风格。
    *   **加载中:** 蓝色和橙色的加载动画，文字 "DEVELOPING..."。
*   **后期制作 (Magic Edit):**
    *   位于下方的白色工具栏，提供文字输入进行 AI 修图。

### C. 页脚
*   模拟爵士乐演出门票存根或西部海报底部的版权信息栏，使用克莱因蓝底色和白色文字。

## 4. AI 服务集成 (保持不变)
*   **`generateSetPhoto`**: 生成核心图片。
*   **`editSetPhoto`**: 图片后期修改。
*   **`getMoviePosterImpression`**: 生成背景氛围图。
